### Run Config

**EN :** 

1. Install Dependency 

   - openbox

   - tint2

   - polybar

   - rofi

   - obmenu-generator dependency

   Optional dependencies
    -  fish
    -  oh my fish 
    -  neofetch
    -  htop
   
2. Copy All folders to `~/.config`

**FA :** 

1. نصب پیش نیاز ها

   - openbox

   - tint2

   - polybar

   - rofi

   - obmenu-generator dependency

   پیش نیاز های اختیاری
    -  fish
    -  oh my fish 
    -  neofetch
    -  htop
   
2. کپی کردن تمامی پوشه ها در ادرس `~/.config`
